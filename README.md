# COVID 19 Tracker


## Project setup

### Install dependencies
```
yarn install
```

### Start project
```
yarn start
```

### Build project
```
yarn build
```


## Development Stack
```
Front-end: React, Material UI, HTML/CSS

Back-end: Node.js

Features: Line Chart, Leaflet Map
```
